# tick_tock ⏰
저희 서비스는 **투두 리스트와 캘린더를 결합한 일정 관리 플랫폼**으로, 개인과 그룹이 더욱 효율적으로 일정을 계획하고 조율할 수 있도록 돕는 것을 목표로 합니다.
기본적으로 **투두 리스트와 캘린더를 연동하여 한눈에 일정과 할 일을 관리**할 수 있으며, **공유 일정 기능과 자동 추천 시스템**을 통해 **팀원, 친구, 가족 등과의 일정 조율을 간편하게 해결**할 수 있습니다.
